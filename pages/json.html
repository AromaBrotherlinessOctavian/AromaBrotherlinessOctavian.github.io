JSON

Синтаксис JSON (JavaScript Object Notation)

Определяется стандартом RFC 8259 (текущая версия). JSON-документы описывают JS-объекты. Комментариев в JSON не предусмотрено.

На самом верхнем уровне, JSON-text'ы содержат некий JSON-value, которых всего несколько типов:

  • Null литерал null, представляющий отсутствие объекта
  • Булеаны литералы true/false - да/нет
  • Число числовые литералы - дробные/целочисленные
  • Строки строчные литералы, содержащие Unicode символы
  • Гетерогенные массивы последовательности JSON-значений (не обязательно одинаковых типов), разделённые запятыми
  • Объекты пары "ключ-значение" (где ключ - строка, а значение - любое JSON-value), разделённые запятыми

JSON-text = *пробел (string | number | object | array | true | false | null) *пробел 


{
    "Image": {
        "Width":  800,
        "Height": 600,
        "Title":  "View from 15th Floor",
        "Thumbnail": {
            "Url":    "http://www.example.com/image/481989943",
            "Height": 125,
            "Width":  100
        },
        "Animated" : false,
        "IDs": [116, 943, 234, 38793]
    }
}

Пример JSON-документа: это объект, его поле Image является объектом, у него есть поля Width, Height, Title, Animated, Thumbnail(тоже объект с полями Url, Height, Width), IDs(массив чисел).
