# AromaBrotherlinessOctavian.github.io

[https://aromabrotherlinessoctavian.github.io](https://aromabrotherlinessoctavian.github.io)

В работе рассказывается про API и в частности конкретно про Web-API, про часто используемые протоколы, про разные категории Web-API и про то, как правильно выбрать сервис.
На сайте используется CSS-фреймворк [Semantic UI](https://semantic-ui.com) (см. [Тест CSS-фреймворка](https://aromabrotherlinessoctavian.github.io/pages/css_framework_showcase.html)) и [jQuery](https://jquery.com)

Основная страница - [pages/index.html](pages/index.html)(есть ещё пустышка [index.html](index.html) которая редиректит на основную страницу). Для локального деплоя можно просто скачать корневую папку и открывать .html файлы в браузере, все пути прописаны как относительные.
